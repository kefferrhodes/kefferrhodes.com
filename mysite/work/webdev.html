Web Design

Keffer Rhodes

(this website)

I designed and built this website using HTML, CSS, and the Bootstrap grid framework.

My goal was to create an extremely simple and easy to navigate portfolio for my creative work. I used a mid/late-century aesthetic. It is a full-screen format with minimal blocking, opting for transparent headers/footers with oversized elements and few defined margins.

Heads & Tails

I designed and built this website using Squarespace. I felt Squarespace was ideal due to the number of pages, complexity of the page elements, and the built-in SEO capabilities. I used the complementary color pallete already used by the company.

The site is designed as a funnel. All paths are designed to lead the visitor to the new client consultation form. The site titles, headings, and paragraph text are written for SEO. The site ranks first or second in most relevant searches.

There is clearly defined blocking to separate sections, wide margins, and an easy to navigate feel.
